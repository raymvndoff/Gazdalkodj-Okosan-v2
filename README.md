# Gazdalkodj-Okosan-v2
Az előző projektem (https://github.com/raymvndoff/Gazdalkodj-Okosan-v1) továbbgondolt verziója
Lényegében helyettünk csinál mindent. A játékosok a képernyőre nyomva tudnak dobni, így a program folyamatosan tudja ki melyik mezőn áll, mikor haladt át olyan mezőn ahol egyenlegét terhelni kell, vagy épp jóvá kell írni bizonyos összeget. Semmilyen manuális számítást nem kell végezni, csak annyit kell eldönteni, amikor egy mezőre lépünk, hogy kártyával vagy készpénzzel szeretnénk fizetni. Teljes mértékben az eredeti társasjáték szabályzataival működik.
